## How to Setup

### Requirements

* gphoto2
  * to take photos from DSLR
  * `brew install gphoto2`
* qrencode
  * to generate QR
  * `brew install qrencode`
* FFmpeg
  * to encode GIF
  * `brew install ffmpeg`
